# react-linkedin-example

這是一個練習專案

`create-react-app 4.0`

`react-router`做路由管理

`react-app-rewired`+`customize-cra`做 webpack 擴充，絕對路徑與動態載入 chunkname

`ant-design` UI

`style-component` CSS in JS
